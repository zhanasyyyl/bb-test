Testing Tools


This video is silent.

Section Directions: Directions are on the top left of each test screen.

Testing Timer: Use the timer to pace yourself. You can hide it until 5 minutes remain.

Highlights & Notes: Click Highlights & Notes then select some text to get started.

Line Reader: Use the line reader to help you focus as you're reading test content.

Calculator: On the SAT Suite of Assessments and certain AP Exams, you'll have access to a calculator.
Some exams have more than one kind.

Reference Sheet: Many exams have a reference sheet where you can find common formulas.

Help: Click on More to access Help content.

Expand Left/Right: Grab the center line to expand either side of the screen.

Zoom: Zoom with keyboard shortcuts on laptops or by pinching on tablets. Quick Tip: Zoom using Control +/- (PC) or Command +/- (Mac).

Mark for Review: Use the bookmark icon to flag any question you want to come back to.

Option Eliminator: On multiple-choice questions, if you think an answer option is wrong, you can cross it out

Question Menu: Navigate to any question in the section.
